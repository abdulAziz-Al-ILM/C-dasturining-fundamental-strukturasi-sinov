Kod yozishni emas, tizim qurishni o'rganing

Dars hikoyasi: "Raqamli Zavod"

Tasavvur qil, biz hozir bitta robot yasayapmiz. U hali o'lik metall bo'lagi. Uni ishga tushirish uchun bizga 4 ta narsa kerak: Asboblar qutisi, zavod nomi, robotning qobig'i va "Start" tugmasi. C# dagi har bir qator aynan shu vazifani bajaradi!

using System;

namespace ZavodApp
{
    class Robot
    {
        static void Main()
        {
            Console.WriteLine("Tizim ON!");
        }
    }
}
1. Asboblar qutisi using System; — Bu robotga gapirish va eshitishni o'rgatuvchi lug'at.
2. Xona nomi namespace Zavod; — Robotimiz qaysi bo'limda saqlanishini bildiradi.
3. Robot tanasi class Robot; — Robotning tashqi korpusi (qobig'i).
4. Quvvat tugmasi Main(); — Robotni o't oldiradigan asosiy nuqta.

⚡ Mantiqiy Sinov

Robotimiz hozir faqat "Tizim ON!" deya oladi. Agar sen unga o'z ismingni ayttirmoqchi bo'lsang, kodning qaysi qismini o'zgartirgan bo'lar eding?

Vazifa: Kodni nusxalab, Console ichidagi yozuvni o'zgartirib ko'r!
